1. Introduction

Bienvenue sur AllôPharmacy. La présente Politique de Confidentialité a pour but de vous expliquer comment nous recueillons, utilisons, protégeons et partageons vos informations personnelles lorsque vous utilisez notre application mobile et nos services, notamment lorsque vous choisissez de vous connecter via Facebook (Facebook Login).

En utilisant l'application AllôPharmacy, vous acceptez les pratiques décrites dans cette politique.

2. Données collectées via Facebook Login

Si vous choisissez de vous connecter avec Facebook, nous recevons de Facebook les informations que vous autorisez :

  • Profil public : prénom, nom et photo de profil.
  • Adresse e‑mail : l'adresse associée à votre compte Facebook.

Note : AllôPharmacy ne demande aucune autorisation pour publier sur votre mur, accéder à votre liste d'amis ou lire vos messages privés.

3. Utilisation de vos données

Les informations recueillies sont utilisées uniquement pour :

  • Création et gestion de compte : identification unique sur la plateforme.
  • Personnalisation : afficher votre nom et photo dans l'application.
  • Communication : vous informer des mises à jour importantes.
  • Fonctionnement : lier recherches et historiques de géolocalisation de pharmacies à votre compte pour un service sur mesure.

4. Partage et protection de vos données

  • Non-revente : nous ne vendons, louons ni commercialisons jamais vos informations personnelles.
  • Sécurité : stockage sécurisé via notre infrastructure Supabase et protocoles de chiffrement standards.
  • Services tiers : partage limité aux prestataires nécessaires au fonctionnement technique, soumis aux mêmes obligations de confidentialité.

5. Suppression des données utilisateurs

Conformément au RGPD et aux exigences Facebook Developer, vous pouvez demander la suppression totale de vos données. Deux options :

Option 1 Directement dans l'application

  1. Ouvrez l'application AllôPharmacy.
  2. Allez dans le menu "Profil" ou "Paramètres".
  3. Cliquez sur "Supprimer mon compte".
  4. Confirmez : vos données seront effacées de nos serveurs.

Option 2 Par e‑mail

Envoyez une demande à support@allopharmacy.com avec l'objet Demande de suppression de données Facebook et l'adresse e‑mail associée à votre compte. Nous traiterons la demande sous 7 jours ouvrés.
